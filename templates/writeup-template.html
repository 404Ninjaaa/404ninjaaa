Machine Name Writeup

Date: November 23, 2025 • Platform: HackTheBox • Difficulty: Easy

Introduction

Brief description of the machine. Mention the OS (Linux/Windows) and the key techniques used (e.g., SQL Injection, Kernel Exploit).

Reconnaissance

Started with an Nmap scan to identify open ports:

nmap -sC -sV -oA nmap/initial 10.10.10.x

The scan results showed port 22 (SSH) and port 80 (HTTP) open.

Exploitation

Web Enumeration

Found a login page at /login.php. Tried default credentials but failed.

Gaining Access

Used SQLMap to dump the database and found a password hash. 

sqlmap -u "http://10.10.10.x/login.php" --forms --dump

Privilege Escalation

After getting a shell as www-data, I ran linpeas.sh and found a SUID binary.

Conclusion

This box was a great exercise in basic web enumeration and standard Linux privilege escalation.
